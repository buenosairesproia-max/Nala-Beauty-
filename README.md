# NALA Beauty Clinic - Dra. Estefania Balboa

## Proyecto
- **Nombre**: NALA Beauty Clinic
- **Objetivo**: Plataforma web premium de medicina estetica
- **Doctora**: Dra. Estefania Maria Belen Balboa - Dermatologa & Master Injector

## URLs
- **Sandbox**: https://3000-iwz56k2x8wfly8389ps40-5185f4aa.sandbox.novita.ai
- **Instagram**: @nala_beautyclinic

## Arquitectura
- **Framework**: Hono (Cloudflare Workers)
- **Build**: Vite
- **Deploy Target**: Cloudflare Pages
- **Tipo**: Single Page Application (SPA)

## Secciones
1. **Home** - Hero con imagenes B&W (click para color) + typing effect + resumen de todas las secciones
2. **Sobre Mi** - Galeria horizontal con auto-scroll, credenciales, Instagram Reels
3. **Tecnologia** - Morpheus8, Lutronic Spectra XT, Scizer, Harmony XL PRO, IA diagnostico
4. **Productos** - Profhilo, Morpheus8 Resurfacing, Nordlys, Radiesse, TETRA PRO
5. **Servicios** - Tratamientos laser, facial integral, corporal, bioestimulacion
6. **Experiencia** - AMWC Awards, congresos Monaco, certificaciones internacionales
7. **Ubicacion** - Interiores de la clinica, arquitectura, ambientes
8. **Contacto** - Formulario, informacion, redes sociales

## Funcionalidades
- Navegacion SPA real (sin scroll infinito)
- Hero con imagenes en B&W que se transforman a color al click
- Efecto de escritura letra por letra en frase principal
- Carruseles horizontales con auto-scroll cada 2 segundos
- Drag/swipe manual en todos los carruseles
- Modal de detalle al hacer click en tarjetas
- Animacion de entrada de bloques en Home
- Menu mobile con overlay
- Formulario de contacto funcional
- 40+ imagenes clasificadas inteligentemente

## Tecnologias/Marcas Representadas
Morpheus8, Lutronic Spectra XT, Scizer, Harmony XL PRO, Nordlys (Candela), Profhilo (IBSA), Radiesse, TETRA PRO (DEKA), EvolveX

## Comandos
```bash
npm run build        # Construir el proyecto
npm run dev          # Servidor de desarrollo
npm run deploy       # Deploy a Cloudflare Pages
```
